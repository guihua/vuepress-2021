Git 清理
===

### 查看项目文件明细
```bash
du -d 1 -h
```

### 清理和回收空间
```bash
# 清理
# [bfg-repo-cleaner](https://rtyley.github.io/bfg-repo-cleaner/)
java -jar bfg-1.11.7.jar -b 1M myrepo.git
# 压缩
rm -rf .git/refs/original/
git reflog expire --expire=now --all
git gc --prune=now
# 提交
git push --all --force
git push --all --tags --force
```
