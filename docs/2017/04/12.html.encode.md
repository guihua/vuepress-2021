html 转义+过滤
===

前端代码导致的 XSS，关键问题在于把数据插入到 DOM 的时候，没有做转义或者过滤。

HTML 转义：
```js
var encodeHtml = function (str) {
    return String(str)
        .replace(/&/g, '&amp;')
        .replace(/"/g, '&quot;')
        .replace(/'/g, '&#39;')
        .replace(/`/g, '&#96;')
        .replace(/</g, '&lt;')
        .replace(/>/g, '&gt;');
};
```
